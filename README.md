# Point-Feature RGB-D SLAM Submission

## Datasets Used

The project used the following datasets:

- `xyz` -> `rgbd_dataset_freiburg1_xyz`
- `room` -> `rgbd_dataset_freiburg1_room`
- `desk2` -> `rgbd_dataset_freiburg1_desk2`
- `360` -> `rgbd_dataset_freiburg1_360`
- `floor` -> `rgbd_dataset_freiburg1_floor`
- `fr2desk` -> `rgbd_dataset_freiburg2_desk`
- `loop` -> local folder `tum_loop`
- `rgbd` -> local folder `tum_rgbd`

Official dataset source:
- TUM RGB-D benchmark: https://cvg.cit.tum.de/data/datasets/rgbd-dataset
