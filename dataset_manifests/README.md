# Dataset Manifests

This folder contains only the small manifest files for the datasets used in the project:

- `rgb.txt`
- `depth.txt`
- `groundtruth.txt`
- `accelerometer.txt`

The raw RGB image folders and raw depth image folders are not included here.

To reproduce the full experiments, download the original datasets from:

- https://cvg.cit.tum.de/data/datasets/rgbd-dataset

Then place the datasets into the same folder names used in this project.
